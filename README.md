## 🚀 نصب و راه‌اندازی

### ۱. کلون کردن پروژه
```bash
git clone https://github.com/your_username/yasha_bot.git
cd yasha_bot
```

### ۲. نصب پکیج‌ها
```bash
pip install -r requirements.txt
```

### ۳. تنظیم Environment Variables
```bash
cp .env.example .env
```
فایل `.env` رو باز کن و مقادیر رو پر کن:

| متغیر | توضیح |
|-------|-------|
| `BOT_TOKEN` | توکن ربات از [@BotFather](https://t.me/BotFather) |
| `OWNER_ID` | آیدی عددی صاحب ربات |
| `BOT_USERNAME` | یوزرنیم ربات (بدون @) |
| `FORCE_CHANNEL_1` | یوزرنیم کانال اول (با @) |
| `FORCE_CHANNEL_ID_1` | آیدی عددی کانال اول |
| `LOGGER_GROUP` | آیدی گروه لاگ |
| `BACKUP_CHAT` | آیدی چت بکاپ |

### ۴. اجرا
```bash
python bot.py
```

## 📁 ساختار پروژه

```
yasha_bot/
├── bot.py              # فایل اصلی ربات
├── config.py           # تنظیمات و متغیرها
├── database.py         # مدیریت دیتابیس SQLite
├── requirements.txt    # پکیج‌های مورد نیاز
├── .env.example        # نمونه فایل تنظیمات
├── .gitignore          # فایل‌های نادیده گرفته شده
└── README.md           # این فایل
```

---

## ⚠️ نکات امنیتی

- **هرگز** فایل `.env` رو توی گیت‌هاب آپلود نکن!
- فایل `.gitignore` این کار رو بلاک می‌کنه، ولی حواست باشه.
- توکن ربات رو فقط توی `.env` بذار.
